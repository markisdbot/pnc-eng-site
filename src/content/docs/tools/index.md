# Tools & Downloads

GOOSE Works provides free engineering tools to help streamline your IEC 61850 and Protection & Control work. These tools are designed by engineers, for engineers.

## Available Tools

### IEC 61850 File Comparison Tool

Compare SCL files (CID, SCD, ICD, SSD) and view color-coded differences. This tool helps validate configuration changes and troubleshoot integration issues.

**Features:**
- Side-by-side comparison of SCL files
- Color-coded differences (added, removed, modified)
- Detailed analysis of IEDs, DataSets, ControlBlocks
- Communication section comparison
- DataTypeTemplates validation

[Open File Comparison Tool →](/tools/iec61850-file-comparison/)

---

### Excel File Comparison Tool

Compare Excel spreadsheets cell-by-cell to identify changes between revisions. Useful for settings files, IO lists, and documentation tracking.

**Features:**
- Cell-by-cell comparison
- Formula vs. value comparison options
- Highlight differences with color coding
- Export difference reports

### Excel GOOSE Mapper Tool

Map GOOSE signals between IEDs using a spreadsheet interface. Simplifies the creation and validation of GOOSE publishing/subscribing configurations.

**Features:**
- Visual mapping of GOOSE publishers and subscribers
- Automatic validation of data types
- Export to SCL format
- Import from existing SCD files

*Download coming soon*

### OpenSCD Desktop

A desktop application for IEC 61850 SCL file editing and system configuration. Based on the open-source OpenSCD project with additional utilities for professional use.

**Features:**
- Visual SCL file editor
- IED configuration management
- DataSet and ControlBlock editing
- Communication configuration
- Substation section editing
