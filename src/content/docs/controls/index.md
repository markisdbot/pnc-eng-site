# Controls

GOOSE Works provides control system design and integration services for substation automation projects. We bridge the gap between protection systems and SCADA, ensuring reliable control and monitoring capabilities.

## Services

### SCADA Integration
- DNP3 and IEC 61850 protocol implementation
- Point list development and mapping
- Control scheme logic design
- Alarm and event configuration

### RTU & PLC Programming
- Remote Terminal Unit (RTU) configuration
- Programmable Logic Controller (PLC) programming
- Control sequences and interlocks
- Local/Remote control schemes

### HMI Development
- Human-Machine Interface design
- One-line diagram development
- Alarm management and annunciation
- Historical data and trending

### Substation Automation
- Integrated automation system design
- Bay controller configuration
- Time synchronization (IRIG-B, PTP)
- Sequence of events (SOE) recording

## Approach

Our control system designs prioritize:
- **Reliability** — Redundant communications and fail-safe designs
- **Security** — Defense-in-depth cybersecurity practices
- **Maintainability** — Clear documentation and standardized configurations
- **Scalability** — Designs that accommodate future expansion
